# 🧪 Receta txAdmin – QBCore + Ox Ecosystem

**Descripción:**
Una receta de **txAdmin lista para producción basada en QBCore**, adaptada con el ecosistema de **Overextended (Ox)**. Este pack proporciona una base **moderna, estable y optimizada** para tu servidor de FiveM con una configuración mínima y máxima flexibilidad.

Esta receta viene preconfigurada para una compatibilidad total con:

* **ox_inventory** (Inventario moderno por metadatos)
* **ox_target** (Sistema de interacción por diana)
* **ox_doorlock** (Gestión de cerraduras y puertas)
* **ox_lib** (Librería de utilidades avanzada)

---

## Qué está incluido

No es solo una base de QBCore; es un **framework de inicio seleccionado** y optimizado para el rendimiento y la experiencia del jugador. Características clave:

* **illenium-appearance** — Reemplazo moderno y potente para `qb-clothing`.
* **cd_easytime** — Sistema de clima y tiempo optimizado (reemplaza a `qb-weathersync`).
* **Sistemas por Defecto de QBCore** — Se mantienen los módulos oficiales de **Policía, Ambulancia y Banca** para mayor estabilidad.
* **community_bridge** — Capa de utilidad compartida para reducir la duplicación de scripts.
* **Preparado para Ox** — Configuración de `server.cfg` optimizada para el inventario de Ox.
* **Iconos de Ítems** — Incluye descarga automática de imágenes para el inventario.
* **Ligero y Limpio** — Se han eliminado más de 35 scripts innecesarios (apartamentos por defecto, trabajos básicos, etc.) para que puedas construir tu servidor desde una base limpia.

---

## Notas de Uso

> ⚠️ Esta receta **no aparece** en el navegador de recetas predeterminado de txAdmin.
> Para instalarla, deberás añadirla manualmente usando la opción **Remote URL** en txAdmin.
> Introduce este enlace (una vez subas el archivo a tu GitHub): `https://raw.githubusercontent.com/ultimorefugio80-code/Receta-Qb-Ox/refs/heads/main/qb_ox.yaml`

Esta receta está diseñada para ser utilizada con [**txAdmin**](https://github.com/tabarra/txAdmin), la herramienta de gestión de servidores FiveM más utilizada.

Para obtener ayuda con la configuración, consulta la [**Documentación Oficial de Recetas de txAdmin**](https://github.com/tabarra/txAdmin/blob/master/docs/recipe.md).
